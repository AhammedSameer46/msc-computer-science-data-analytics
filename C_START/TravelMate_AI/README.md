# TravelMate AI

A simple AI-powered travel assistant built using Python and the Groq API.

## Features

- Check weather for a city
- Search available flights
- Recommend hotels
- Suggest tourist attractions
- Demonstrates AI tool calling

## Technologies Used

- Python
- Groq API
- OpenAI Python SDK
- python-dotenv
